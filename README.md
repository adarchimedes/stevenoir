## Deploy Options

### Option 1: GitHub Pages (FREE)
1. Create repo: `github.com/jakezelinger/stevenoir`
2. Push `website/` folder
3. Enable Pages in settings
4. Live at: `stevenoir.github.io`

### Option 2: Custom Domain (PAID)
1. Buy domain: `stevenoir.ai` or `thestevening.com`
2. Point DNS to GitHub Pages OR
3. Use Vercel/Netlify (free tier available)
4. Deploy in seconds

### Option 3: Heroku/Railway (FREE)
1. Add minimal Python server
2. Deploy website
3. Custom subdomain
